集合的学习总结

Collection

集合接口，其下主要子类接口为List和Set。

List

** 特征 **有序集合，允许元素重复，有索引可以使用普通的for循环。

  • ArrayList
    底层是数组，查询快，增删慢，多线程。
  • LinkedList
    底层是链表结构，查询慢，增删快，多线程。
    注意不能使用多态。
  • Vector
    底层是数组，单线程。
    已被ArrayList替代。
Set

** 特征 **不允许元素重复，没有索引，不可以使用普通的for循环遍历。

  • HashSet
    无序集合，存储元素和取出元素的顺序可能不一致。
    底层是一个哈希表结构。
    查询速度非常快。
  • LinkedHashSet
    底层是哈希表，但多了一条链表用来记录存储顺序。
    保证元素有序。
  • TreeSet
